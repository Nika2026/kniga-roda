# kniga-roda
Создайте материальное наследие своего Рода за 8 месяцев. Курс «Книга Рода» — это глубокое погружение в историю семьи и технологии современного книгоиздания. Под личным наставничеством вы оцифруете фото, сверстаете макет, создадите уникальный шрифт и соберете книгу в твердом переплете. Выберите свой тариф заботы.
